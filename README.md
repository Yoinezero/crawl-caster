# Crawl caster

An experiment app to try web-scrapping, web-crawling and content aggregation.

For now planning to store data in datalake (PC memory or Cloud buckets).

Stack:
- python 3.11
- fastapi
- uvicorn
- aiohttp
- beautifulsoup4

Dependencies managed by Poetry. To install them use:

```
$ pip install poetry
$ poetry install
```
